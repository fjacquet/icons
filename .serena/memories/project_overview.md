# Project Overview: icons

## Purpose

React Icons to SVG Exporter — a static React web app hosted on GitHub Pages. Users browse React-based icon components, customize them (size, color, background), and download standalone SVG files or ZIP archives for use in presentations (PowerPoint, Keynote, Google Slides).

**Current state**: Fresh Create React App scaffold. MVP features are not yet implemented — the app currently shows only the default CRA template.

## Tech Stack

- **Frontend**: React 19 + TypeScript (migration from JS planned per constitution)
- **Build tool**: Create React App (react-scripts 5.0.1)
- **Testing**: @testing-library/react, @testing-library/jest-dom
- **Hosting**: GitHub Pages (static deployment)
- **CI**: GitHub Actions (Node 18.x, 20.x, 22.x matrix)
